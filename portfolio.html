Interactive Sketch

With the use of Processing, I created a code where the mouse would form a trail of ellipses wherever the mouse moved, and if you clicked with the mouse, the ellipses would grow. I wanted to create my own sketch canvas with this code, making the background completely black and the ellipses white so the person using the code could try and draw something of their own.
